Java Collection Framework(JCF)

Posted on 2019-02-01 | Edited on 2019-02-06

概述

集合，集合可以用于将多个数据组织到一起或将数据关联。集合的作用就是保存、盛装其他数据，因此集合类也可以被称为容器类。

java的集合类只能用于存放引用类型的数据，而不能存放基本数据类型。对于基本类型想要使用集合存放，必须将其包装成对象类型。

java.util.ArrayList实现的接口

子类

java.util.ArrayList 类提供了可调整大小的数组，并实现了List接口。ArrayList的特点：

  • 实现了所有可选的列表操作，并且允许所有元素，包括空值null。

  • 提供了一些方法来操作内部用来存储列表的数组的大小。

  • 相较于LinkedList实现的常数因子较低。

calculateCapacity(Object[] elementData, int minCapacity)

该方法用于处理以默认大小初始化的elementData未添加元素的情况下的capacity计算问题。首先判断elementData是否是DEFAULTCAPACITY_EMPTY_ELEMENTDATA，如果是则取默认capacity与minCapacity中较大的，如果不是则返回minCapacity。
