矩阵链乘法问题

矩阵链乘法是可用动态规划解决的最佳化问题。

问题描述

给定一个$n$个矩阵的序列（矩阵链） < $ A_1, A_2, \cdot\cdot\cdot, A_n $> ，我们希望计算它们的乘积

为了计算公式$\eqref {eq:公式1}$，我们可以先用括号明确计算次序，然后利用标准矩阵相乘算法进行计算。矩阵乘法是满足结合律的，因此任何加括号的方法都会得到相同的计算结果。

我们称具有如下性质的矩阵乘积链为完全括号化的($fully\, parenthesized$)：它是单一矩阵，或者是两个完全括号化的矩阵乘积链的积，且已外加括号。例如对于4个矩阵的序列，有如下完全括号化的方法：

对矩阵链加括号的方法会对乘积运算的代价产生巨大影响。两个矩阵相乘的代价取决于相乘的两个矩阵的矩阵规模，若矩阵$A, B$的规模分别为$p\times q, q\times r$，则计算矩阵乘积的代价由进行的标量乘法所决定，即：$pqr$.

矩阵 $ A_i $ 的规模为 $ p_{i-1} \times p_i （1\leq i \leq n）$，求完全括号化方案，使得计算乘积 $ A_1, A_2, \cdot\cdot\cdot, A_n $ 所需标量乘法次数最少。

Solution

Approach 1：递归解法

使用符号$A_{i, \cdot\cdot\cdot j}(i\leq j)$表示$A_iA_{i+1}\cdot\cdot\cdot A_j$乘积的结果矩阵。令$m[i,j]$表示计算矩阵$A_{i, \cdot\cdot\cdot j}(i\leq j)$所需的标量乘法次数的最小值，那么原问题的解-计算$A_{1, \cdot\cdot\cdot n}$所需的最低代价就是$m[1, n]$。

进行如下的对于$m[i, j]$的递归定义。对于$i = j$时的平凡问题，矩阵链只包含唯一的矩阵$A_{i, \cdot\cdot\cdot i} = A_i$，因此不需要做任何的标量乘法运算。对于所有的$i = 1, 2\cdots, n m[i, j] = 0$。当$i<j$时可以使用最优子结构来计算$m[i,j]$。
