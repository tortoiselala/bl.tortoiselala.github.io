Elementery Graph Algorithms Introduction to Algorithms

Posted on 2018-11-16

图

概念

稀疏图：边的条数$|E|$远远小于$|V^2|$的图

稠密图：$|E|$接近$|V^2|$的图

图的表示

对于图$G=(V,E)$，可以用两种标准方法表示。

  • 邻接表
  • 邻接矩阵

两种方式都可以表示无向图和有向图。在表示稀疏图时我们更趋向于使用邻接矩阵表示法，而在表示稠密图时邻接矩阵则更为方便。此外，邻接矩阵表示法允许我们快速判断两点之间是否有边相连。但是由于临界矩阵的建立要比邻接矩阵更为简单，所以大多数情况下，我们使用临界矩阵。
